Proyecto Web - Sistema de Gestión de Trabajadores

- Descripción

Este proyecto consiste en el desarrollo de una aplicación web para la gestión de trabajadores dentro de un departamento de Recursos Humanos, utilizando HTML5, Bootstrap y JavaScript. Permite añadir empleados y visualizar un resumen dinámico por departamentos.

- Estructura del proyecto

El proyecto está compuesto por los siguientes archivos y carpetas:

index.html → Página principal de la aplicación

js/script.js → Lógica de la aplicación en JavaScript
